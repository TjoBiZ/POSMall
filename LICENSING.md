# POSMall Licensing Map

This repository contains POSMall product work, demo materials, fork-origin
portions from OFFLINE.Mall by OFFLINE GmbH, and third-party dependencies. It is
publicly visible, but original POSMall additions are not released for standalone
resale, repackaging or redistribution.

## Original POSMall additions

Original POSMall additions created by KodZero / POSMall contributors are governed
by:

```text

```

This includes PostgreSQL-first architecture additions, API and GraphQL
additions, product implementation fragments, UI/business-flow additions,
documentation, examples, benchmark materials, seed examples, demo business
content, storefront/theme integration materials and other original POSMall work,
unless a file-level notice says otherwise.

In short: you may use and modify these POSMall additions for personal projects,
internal projects, client implementation/support work and operating a website or
application. You may not resell, sublicense, redistribute, publish, extract or
package them as a standalone product, marketplace item, plugin, theme, starter
kit, source package, API example pack, benchmark package or competing package.

## Fork-origin portions

This repository is published from a fork of OFFLINE.Mall by OFFLINE GmbH.
Fork-origin portions keep their upstream notice and license continuity.

The fork-origin notice is preserved in:

```text
THIRD-PARTY-NOTICES/OFFLINE-MALL-LICENSE.md
```

That upstream notice applies only to fork-origin code and upstream-covered
portions. It does not license original POSMall additions as standalone resale
material.

POSMall does not claim ownership of fork-origin code. POSMall claims ownership
and applies its non-resale terms only to original POSMall additions and
POSMall-owned materials.

If a file contains an explicit license notice, that file-level notice controls
that file. Otherwise, use this document to determine which license applies
based on the file's origin and role.
